# Frontend (Quasar)

Este proyecto es un frontend construido con [Quasar Framework](https://quasar.dev).

## Requisitos previos

* Tener instalado [Node.js](https://nodejs.org/) (v16 o superior)

## Ejecutar en desarrollo

1. Clonar el repositorio y entra en el directorio del frontend:
```zsh
git clone https://github.com/yes-lonso/DeviceLink.git
cd DeviceLink/frontend
```

2. Instala las dependencias:
```zsh
npm install
```

3. Inicia el servidor de desarrollo:
```zsh
quasar dev
```

### Compila y minimiza para producción
```
quasar build
```

### Revisa y corrige los archivos
```
npm run lint
```

### Personalizar la configuración
Ver [Configurando quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).
